Question-1: What are the different ways to select an element in the DOM?

JavaScript selects DOM elements using: getElementById(id):It finds a single element by its unique ID. getElementsByClassName(class): Finds multiple elements with the same class. getElementsByTagName(tag): Finds multiple elements with the same tag-line (e.g., p, div ). querySelector(selector): Finds the first element matching a CSS selector. querySelectorAll(selector): Finds all elements matching a CSS selector. getElementsByName(name): Finds form elements by their name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

We use innerHTML when you need to work with HTML markup. and textContent when you only need to work with the raw text content, farther more want to be sure we are getting all of the text inside of an element. lastly we use innerText when you want the rendered text content, but be aware that it might not include all text.

Question-3: What is event delegation in the DOM?

Event delegation means putting one event listener on a parent element instead of many on its children. When a child is clicked, the event "bubbles up" to the parent, where the listener checks which child was the real target. This saves memory and makes it easier to handle events, especially with lots of children or when new ones are added later.

Question-4: What is event bubbling in the DOM?

When we click a button inside a box. The "click" event doesn't just stay on the button. It travels "upwards" through the box, then any boxes around it, and so on. That's event bubbling: events start at the clicked element and then "bubble" up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript manipulates the DOM by: Creating: document.createElement() makes new elements. Adding: appendChild(), insertBefore(), or insertAdjacentElement() places them into the HTML. Removing: removeChild() or remove() deletes elements. Adding text: textContent adds plain text inside elements.
